[−][src]Module serato_tags::tag::markers

The Serato Markers_ tag stores information about the first 5 Cues, 9 Loops and the track color.

This is redundant with some of the information from the Serato Markers2 tag. Serato will prefer information from Serato Markers_ if it's present.
